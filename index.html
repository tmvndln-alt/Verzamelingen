AI-herkenning

Vul hier je gratis Google Gemini API-key in om foto's automatisch te laten herkennen. Je haalt een key op via aistudio.google.com/apikey — inloggen met een Google-account, geen creditcard nodig.

De key wordt alleen lokaal in je browser bewaard (localStorage) en nergens anders naartoe gestuurd dan rechtstreeks naar Google.

Synchroniseren tussen apparaten

Wil je dezelfde collectie zien op iPad én iPhone? Maak eenmalig een gratis Firestore-database aan via console.firebase.google.com:

  1. Nieuw project aanmaken (naam maakt niet uit)
  2. Linksmenu → Databases & Storage → Firestore → "Create database" → kies een regio dichtbij (bv. europe-west) → start in testmodus
  3. Tabblad "Rules" → vervang de inhoud door de open regels hieronder → Publiceren
  4. ⚙ Project settings → onderaan "Your apps" → webapp toevoegen (``-icoon) → kopieer het hele firebaseConfig-object
  5. Plak dat object hieronder en bewaar. Doe dit op elk apparaat met dezelfde config.
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

Let op: deze regels maken de database open voor iedereen die de configuratie kent. Voor een privéverzameling is dat een aanvaardbaar risico zolang je de config niet deelt, maar het is geen echte beveiliging.
