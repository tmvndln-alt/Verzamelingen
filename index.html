Synchroniseren tussen apparaten

Gebruik dezelfde Firebase-configuratie als je Vondstenboek (of maak een nieuw project aan via console.firebase.google.com → Databases & Storage → Firestore → Create database → testmodus → Rules aanpassen naar onderstaande → webapp toevoegen voor de config).

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
